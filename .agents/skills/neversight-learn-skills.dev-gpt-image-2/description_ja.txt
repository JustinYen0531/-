RunComfy 上の OpenAI GPT Image 2 (ChatGPT Images 2.0) を使用して画像を生成および編集します。 GPT Image 2 の長所 (埋め込みテキスト、ロゴ、多言語タイポグラフィ、指示の精度)、その 3 つの固定サイズ、保存付き編集言語、および代わりに兄弟 (Flux 2 / Nano Banana Pro / Seedream) にルーティングする場合について文書化します。ローカルの RunComfy CLI を介して `runco​​mfy run openai/gpt-image-2/text-to-image` または `/edit` を呼び出します。 「gpt image 2」、「gpt-image-2」、「ChatGPT Images 2」、「image 2」、またはこのモデルで生成または編集するための明示的な要求でトリガーされます。
