Создавайте и редактируйте изображения с помощью OpenAI GPT Image 2 (ChatGPT Images 2.0) в RunComfy. Документирует сильные стороны GPT Image 2 (встроенный текст, логотипы, многоязычная типографика, точность инструкций), его 3 фиксированных размера, язык редактирования с сохранением и время перенаправления на родственную версию (Flux 2 / Nano Banana Pro / Seedream). Вызывает `runcomfy run openai/gpt-image-2/text-to-image` или `/edit` через локальный интерфейс командной строки RunComfy. Запускается по «изображению gpt 2», «gpt-image-2», «изображениям ChatGPT 2», «изображению 2» или любому явному запросу на создание или редактирование с помощью этой модели.
