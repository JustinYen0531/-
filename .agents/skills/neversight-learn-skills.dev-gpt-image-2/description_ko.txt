RunComfy에서 OpenAI GPT Image 2(ChatGPT Images 2.0)를 사용하여 이미지를 생성하고 편집하세요. GPT Image 2의 장점(삽입된 텍스트, 로고, 다국어 타이포그래피, 지시 정밀도), 3가지 고정 크기, 보존으로 편집 언어, 대신 형제(Flux 2/Nano Banana Pro/Seedream)로 라우팅하는 시기를 문서화합니다. 로컬 RunComfy CLI를 통해 `runcomfy run openai/gpt-image-2/text-to-image` 또는 `/edit`를 호출합니다. "gpt image 2", "gpt-image-2", "ChatGPT Images 2", "image 2" 또는 이 모델을 사용하여 생성 또는 편집하라는 명시적인 요청을 트리거합니다.
