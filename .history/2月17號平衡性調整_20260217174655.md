# 2月17號平衡性調整（原本 → 調整後）

以下只整理你用引號標示的調整項目，格式為「原本 → 調整後」。

## 1. 將軍

### A) 主動造成傷害進化
- Tier 1（攻擊增傷）：`原本有攻擊增傷（4→6）` → `移除攻擊增傷`
- Tier 1（踩雷易傷）：`原本為固定 +2` → `改為每次 +1，最多疊到 +2（死亡重置）`
- Tier 2（受傷者移動耗能）：`原本為加倍` → `改為每格 +2`
- Tier 3-1（持旗攻擊回血）：`2` → `4`
- Tier 3-1（持旗攻擊能耗）：`8` → `6`
- Tier 3-2（擊退上限）：`1 格` → `最多 2 格（將軍最多前進到 3 格）`

### B) 背旗進化
- Tier 2（旗幟範圍減傷）：`30%` → `25%`
- Tier 3-1（背旗移動每格能耗）：`5` → `4`
- Tier 3-2（領域入場傷害）：`1` → `2`
- Tier 3-2（回合結算傷害）：`2` → `4`

## 2. 掃雷

### A) 標記地雷進化
- Tier 1（建築能耗）：`5` → `6`
- Tier 3-1（建築能耗）：`6` → `5`
- Tier 3-2（行為性質）：`引爆` → `拆除`
- Tier 3-2（附帶傷害）：`無` → `對敵方單位造成 3 點`

### B) 安全回合進化
- Tier 2（能耗減免）：`原本有減免` → `取消減免（維持 5 點）`
- Tier 3-1（能耗）：`5` → `4`
- Tier 3-2（能耗增加）：`原本有增加` → `取消增加`

## 3. 遊俠

### A) 移動進化

- Tier 3-2（遊俠傳送能耗）：`5` → `3`

### B) 搬運地雷進化
- Tier 1（拾取範圍）：`原本距離3X3 ` → `曼哈頓距離 2`
- Tier 3-2（投放範圍）：`近距離` → `曼哈頓距離 2`
- Tier 3-2（投放能耗）：`原本較高` → `5`

## 6. 職業基礎設定補充

### 掃雷（HP:14）
- 基礎 1x1 標記能耗：`原本固定 4` → `同一回合前 2 次為 3，第 3 次起為 4`
- 標記回饋：`原本非即時` → `改為立即顯示成功 / 失敗`
- 標記失效條件：`原本可持續` → `掃雷者移動後失效`

### 製雷（HP:12）

- 地雷能耗規則：`原本可能套用統一值` → `僅普通地雷為 5，其餘地雷維持各自原本能耗`

### 解雷（HP:18）
- 主動解除（已標記可無傷）能耗：`原本較高` → `2`
