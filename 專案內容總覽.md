# 地雷棋專案內容總覽

## 專案資訊
- 名稱：`mine-chess`
- 技術：React + TypeScript + Vite + TailwindCSS
- 核心玩法：回合制、地雷、進化、奪旗

## 主要目錄
- `src/`：主要遊戲程式碼
- `public/`：靜態資源（音樂、背景圖）
- `dist/`、`test_dist/`、`temp_build/`：建置輸出/測試輸出
- `backups/`、`.history/`：備份與歷史檔
- `testsprite_tests/`：測試相關資料

## 主要程式入口
- `index.html`
- `src/main.tsx`
- `src/App.tsx`

## src 核心檔案
- `src/gameEngine.ts`：戰鬥/地雷/規則核心計算
- `src/gameInit.ts`：地圖與初始狀態建立
- `src/gameHelpers.tsx`：UI/規則輔助函式
- `src/constants.ts`：常數設定
- `src/types.ts`：主要型別定義
- `src/i18n.ts`：多語系文字

## hooks
- `src/hooks/useGameAI.ts`：PvE AI 行動邏輯
- `src/hooks/useGameLoop.ts`：回合/計時/鍵盤控制循環
- `src/hooks/usePlayerActions.ts`：玩家與單位動作（移動、攻擊、放雷等）

## components
- `src/components/GameField.tsx`：棋盤與格子顯示
- `src/components/GridCell.tsx`、`src/components/GridCell.css`：單格呈現與效果
- `src/components/ControlPanel.tsx`：底部控制面板
- `src/components/UnitInfoPanel.tsx`：單位選擇欄
- `src/components/GameHeader.tsx`、`src/components/GameModals.tsx`：上方資訊與彈窗
- `src/components/EvolutionTree.tsx`：進化樹 UI
- `src/components/SandboxPanel.tsx`、`src/components/Tutorial.tsx`：沙盒與教學

## 其他重要檔案
- `README.md`：專案說明
- `src/GameGDD.md`：遊戲設計文件
- `EVOLUTION_IMPLEMENTATION_GUIDE.md`：進化系統實作指南
- `task點.md`：目前開發任務與分析紀錄

## 執行指令
- 開發：`npm run dev`
- 建置：`npm run build`
- 預覽：`npm run preview`
- 指定埠預覽：`npm run serve`

## 備註
- 專案內有備份/舊版本檔案（如 `App備份.tsx`、`Old App.tsx`），目前主要運行檔是 `src/App.tsx`。
