# 地雷棋 (Mine Chess)

一款結合地雷、進化與奪旗機制的戰略回合制遊戲。

## 功能特色

- **多種單位類型**：將軍、掃雷兵、遊俠、製雷兵、拆彈兵
- **進化系統**：每個單位都有兩條進化路線
- **地雷機制**：策略性地放置和觸發地雷
- **奪旗玩法**：將敵方旗幟帶回自己基地獲勝
- **AI對戰**：支援人機對戰模式
- **多語言支持**：繁體中文、簡體中文、英文

## 遊戲規則

### 基本玩法
1. **佈陣階段**：調整單位位置，放置初始地雷（最多3顆）
2. **策劃階段**：觀察戰場，制定策略
3. **行動階段**：輪流移動單位、攻擊、使用技能

### 勝利條件
將敵方旗幟帶回自己的基地即可獲勝。

### 單位介紹
- **將軍**：可攻擊敵軍，能拾取旗幟
- **掃雷兵**：可掃描地雷位置
- **遊俠**：移動成本低，可搬運地雷
- **製雷兵**：可製造地雷
- **拆彈兵**：受到地雷傷害減半

## 技術棧

- **React 18** + **TypeScript**
- **Vite** - 快速開發構建工具
- **Tailwind CSS** - 樣式框架
- **Lucide React** - 圖標庫

## 安裝與運行

### 前置需求
- Node.js 16+ 
- npm 或 yarn

### 安裝依賴
```bash
npm install
```

### 開發模式
```bash
npm run dev
```
遊戲將在 http://localhost:3000 啟動

### 構建生產版本
```bash
npm run build
```

### 預覽生產版本
```bash
npm run preview
```

## 開發說明

### 項目結構
```
src/
├── components/          # React 組件
│   └── GridCell.tsx    # 遊戲格子組件
├── types.ts            # TypeScript 類型定義
├── constants.ts        # 遊戲常數配置
├── i18n.ts            # 國際化翻譯
├── App.tsx            # 主應用組件
└── main.tsx           # 應用入口
```

### 主要特性
- **狀態管理**：使用 React Hooks 管理複雜遊戲狀態
- **性能優化**：使用 useMemo 和 React.memo 優化渲染
- **響應式設計**：適配不同屏幕尺寸
- **類型安全**：完整的 TypeScript 類型定義
